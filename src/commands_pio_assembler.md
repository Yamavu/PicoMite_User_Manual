### PIO assembler programs

Commands for writing in-line PIO assembler programs. In all cases the parameters are unquoted string literals. Variables are not allowed.

- .PROGRAM name
- .END PROGRAM [list]
- .SIDE SET n
- .LABEL label
- .LINE n or NEXT
- .WRAP n
- .WRAP TARGET n
- JMP params
- WAIT params
- IN params
- OUT params
- PUSH params
- PULL params
- MOV params
- NOP params
- IRQ SET params
- IRQ WAIT params
- IRQ CLEAR params
- IRQ NOWAIT params
- IRQ params
- SET params
