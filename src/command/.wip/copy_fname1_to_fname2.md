## COPY fname1$ TO fname2$

Copy a file from ‘fname1$’ to ‘fname2$’. Both are strings.A directory path can be used in both 'fname$' and 'fname$'. If the paths differthe file specified in 'fname$' will be copied to the path specified in 'fname2$'with the file name as specified. The filenames can include the drivespecification in the case that you are copying to and or from the non-activedrive (see the DRIVE command)
